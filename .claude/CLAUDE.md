# Spacebot

Rust + Tauri desktop app with a React/TypeScript frontend.

## Commands

- `bun run lint` — lint and format check (biome)
- `bun run biome check --write` — auto-fix lint/format
- `bun run typecheck` — TypeScript type check
- `bun test` — run tests
- `cargo build` — build Rust backend
- `cargo tauri dev` — run in dev mode
